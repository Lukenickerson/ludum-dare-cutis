Cutis (latin for "skin") for Ludum Dare

What is Cutis?

Cutis (latin for "skin") is a way to re-skin the Ludum Dare website, home of the largest global game jam community. People all have different opinions on how they would like the Ludum Dare site to look, so Cutis will give the choice to the users.

How does it work?

In order for it to work, a JavaScript file needs to run in your browser while you're at the Ludum Dare site. There are a couple of ways to do this: One way is to use a browser extension. There is an experimental version of the extension available, but the final version is not yet published.

The best solution is if the cutis.js file is included in Ludum Dare's main theme - but for that we'll have to ask Mike (aka. @mikekasprzak, aka. PoV) to add it!

To give it a try, mouse-over the "SKINS" box at the bottom right of this page and select a skin.

How can I contribute?

You can make your own skins and share them with the community. Please read the CONTRIBUTING document for more information.
